### Требования (!!!):
#### python=3.10.0
#### aiogram=2.25.1
####
### Запуск:
#### 1. Установите зависимости :
```
pip install -r requirements.txt
```
#### 2. Создайте файл .env :
``` 
BOT_TOKEN=<токен_бота>
ACCUWEATHER_API_KEY=<api_key>
```
#### 3. Запустите
####
#### P.S. Названия городов нужно вводить на русском языке
